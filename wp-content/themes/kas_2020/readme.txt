### Renaming:
• Rename root directory and 'theme-name' folder
• in theme-name/webpack.config.js, change 'theme-name' on line 68
• in theme-name/style.css, change 'theme-name' on line 2
• in theme-name/package.json, change 'Theme Name' on line 2

Installing webpack
• In Terminal, cd into the theme folder
• run: ```npm install``` then ```npm run watch```
